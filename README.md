# ThinkBoard - Collaborative Whiteboard & Knowledge Base

A modern, feature-rich collaborative workspace that combines a dynamic whiteboard with structured knowledge management.

## Features

### 🧠 Knowledge Base (RAG-Powered)
- **Intelligent Search**: Find information instantly using natural language queries.
- **Document Upload**: Upload PDFs, text files, and other documents to your knowledge base.
- **Contextual Retrieval**: AI answers are powered by your uploaded knowledge, ensuring relevance and accuracy.

### 🎨 Whiteboard
- **Real-Time Collaboration**: Work together on the same canvas in real-time.
- **Shape Manipulation**: Drag, resize, and edit shapes (boxes, circles, etc.) with ease.
- **Sticky Notes**: Add notes to the board for quick ideas and reminders.
- **Drawing Tools**: Freehand drawing with adjustable brush size and color.
- **Text Editing**: Create and edit text blocks directly on the board.
- **Undo/Redo**: Comprehensive history tracking for all actions.
- **Zoom & Pan**: Navigate large boards effortlessly.

### 📁 Dashboard & Workspace Management
- **Workspace Creation**: Create multiple isolated workspaces for different projects or teams.
- **Project Organization**: Manage your boards and documents within projects.
- **Real-Time Stats**: View live statistics for online users and active boards.

### 🔐 Authentication & Security
- **Multi-Tenant Security**: Complete data isolation between workspaces.
- **Email/Password Login**: Secure user authentication.
- **Role-Based Access** (Soon): Future support for Admin, Moderator, and User roles.

### 🚀 Tech Stack
- **Frontend**: Next.js 16, TypeScript, React 19, Tailwind CSS
- **State Management**: Redux Toolkit
- **Styling**: Radix UI, Lucide React, Tailwind CSS
- **Backend**: Node.js, Express.js, Prisma
- **Database**: PostgreSQL
- **Real-Time**: Socket.IO
- **AI/ML**: LangChain, Ollama
- **File Storage**: Cloudinary
